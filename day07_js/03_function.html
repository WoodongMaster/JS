함수(Function) : 특정 기능을 수행하기 위한 코드 - 호출되어야만 작동

  • function 키워드를 이용하여 선언 function [함수명]{}
  • 함수명은 기능을 압축설명하는 의미로 작성하는게 좋음
  • function [함수명] () => () 괄호안에는 파라미터(매개변수)를 선택적으로 선언 가능
  • 자바에서는 파라미터의 타입을 지정해줘야했지만
    자바스크립트에서는 파라미터 타입의 형식 자유도가 높음 (let으로 지정하면 다받음)
  • function [함수명](){ 로직작성 }
  • return은 함수의 실행 종료 역할을 함과 동시에 특정 결과값을 호출한 위치로 반환
  • 외부에서 선언된 함수 (function.js)파일 연결방법
    script src = "파일명" script

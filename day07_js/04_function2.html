함수의 파라미터(매개변수)는 변수명을 지정해서 함수 영역 내부에 전달하는 역할을 한다.

  • 파라미터의 개수는 제한이 없으나 많으면 관리가 힘듬
  • 파라미터 타입은 제한이 없지만 전달된 타입을 그대로 받아서 사용
  • 파라미터는 전달되는 값의 순서, 개수, 유무를 잘 확인해야함
  • 파라미터의 변수명을 지정할 때 외부의 변수명과 중복되어도 영향은 없음
    but 가독성을 위해 중복된 변수명 사용을 지양해야함
  • 함수 내부에서 선언된 변수는 함수 내부에서만 사용 가능, 외부에선 사용 불가
  • 함수는 리턴 유무에 따라 값을 돌려 받을 수 있고, 함수 실행의 종료시점을 정할 수 있음.

함수는 리턴 유무에 따라 값을 돌려 받을 수 있고, 함수 실행의 종료시점을 정할 수 있음.

화살표 함수(Arrow Function) : ES6부터 사용가능한 함수정의법

가독성은 낮으나 코드량을 줄일 수 있음. 표현식 함수처럼 변수화를 사용할 수 있음.

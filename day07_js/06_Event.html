이벤트 객체는 전역객체로써 모든 태그에 객체를 참조시킬 수 있다.

  • a태그와 button 태그에 이벤트를 주로 참조시켜,
    클릭에 대한 반응을 만들때 사용
  • a태그는 href 속성에 의해 자체 이동이 구현되어 있어서 이동을 막는 코드 필요
  • 이벤트 객체를 부여하는 방식3가지. 각자 목적과 방식이 다름
  • 화면 출력 영역 = document 영역(객체) 라고 함
  • 태그 = html element, 속성 = attribute, 일반텍스트 = textNode 라고 함
  • html 태그로 출력 : innerHTML이라는 property 사용
  • text로 출력 : innerText라는 property 사용
  • EventListener로 출력 : .addEventListener를 사용
  • 객체의 값을 가져올 때 (id) : getElementByid('[id명]')
  • 객체의 값을 가져올때 (class) : querySelector('selector')

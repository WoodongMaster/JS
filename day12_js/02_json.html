JSON(JavaScript Object Notation) : 자바스크립트 객체 표기법(XML 대체 사용)

  • 데이터 요청 및 응답처리에 필요한 데이터의 표준 형식을 제공하는 타입
  • 백엔드(서버) 언어의 종류에 상관없이 데이터 송수신 단일 형식으로 통일
  • 데이터의 전송은 전부 String 타입으로 이루어짐
  • 서버로 데이터를 보낼때 > json to string > JSON.stringify(jsObject);
  • 서버한테 데이터를 받을때 > string to json > JSON.parse(stringData);

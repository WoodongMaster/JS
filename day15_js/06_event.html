06_event (DOM Event) document 영역에서 사용되는 이벤트를 정의 마우스, 키보드, 실행 등등 이벤트의 종류가 매우 많기 때문에 웹에서 사용하는 이벤트를 기준으로 학습 마우스 이벤트 : click(), focus(), blur(), input(), change(), mouseover(), mouseout(), select(), scroll(), load(), drag() ⇒ drop() … 키보드 이벤트 : event.key ⇒ keyDown(), keyPress(), keyUp() [key를 떼는 순간?] 기타 이벤트 관련 메서드 : preventDefault() -> 이벤트 취소 , target, submit() addEventListener(’eventName’, callback Function) 메서드를 사용하여 구현하는 것을 권장한다. 메서드를 태그 속성이나, 엘리먼트에 직접 부여하는 방식도 있음 addEventListener를 권장하는 이유 : 같은 엘리먼트에 여러 이벤트를 중복 부여 가능, 적용된 이벤트를 선택적 제거가능 [이미지에 마우스를 올리면 다른 그림으로 변경] [마우스가 해당 영역을 벗어나면 기존 그림으로 다시 변경] DOM Event

가장 많이 다루는 데이터 타입 : string

  • 문자열 데이터는 index(0부터 시작)를 가지고 있음
  • 문자열 데이터 값 내에서는 개행(br) 하는것을 지양해야 함
  • new String(); 객체 타입으로 선언은 가능하나 잘 안씀
  • length : 문자열의 총 길이 => 문자열 마지막 index는 length-1
  • 문자열 사이에 스크립트 글자 삽입가능 (근데 \든 뭐든 써야함)

문자관련 메서드 : java에서 사용하는 string 메서드와 비슷 (다른 언어도 비슷)

  • 문자 추출 관련 메서드 : substring(), substr(), slice()
  • 문자 대체 메서드 : replace(oldStr, newStr) : 첫번째 매칭되는 문자만 + 대소문자 구분
  • trim() : 문자열 양측 공백제거
  • padStart(목표문자길이, 채울문자) : 앞쪽에서부터 / padEnd(목표문자길이, 채울문자) : 뒤쪽에서부터
    이미 3글자(abc)가 있는데 (5, '*')을 주면 **abc가 됨 abc**
  • charAt(index) : 해당 index 문자 추출
  • indexOf("str") : 문자열의 위치를 반환
  • startWith("str") / endWith("str") : 지정한 문자열(str)이 문자열에서 시작/끝에 존재하는지 확인하는 메서드 return -> true /false

자바스크립트의 데이터 타입의 확장 개념 집합데이터를 다룰 수 있는 객체

  • set은 중복데이터 허용 x : 중복데이터가 들어오면 체크 후 하나만 남김
  • set() : 파라미터는 주로 배열을 사용, 배열의 index와 value를 활용하여 집합데이터 생성
  • set 전용 메서드 존재 (찾아봐야함 ㅋㅋ)
  • new set() or new set(array) : set 객체의 생성 및 초기화
  • set => s : s.add(value) 데이터 추가
  • s.has(value) : value의 존재 여부를 true / false 로 리턴
  • s.forEach(callback function) : set 순환
  • s.entries(), s.keys(), s.values() : 리턴 타입은 SetIterator => 별도의 메서드 사용해야 함
  • s.size : set의 개수

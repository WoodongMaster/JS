Map : key/value 형태의 데이터를 한개의 배열로 만들어서 다시 배열에 저장

  • 선언 및 초기화 : new Map(), new Map(['key1', 'value1'],['key2', 'value2'], ...)
  • 기본적으로 중첩구조이기 때문에 key 중복 확인 필수
  • key를 식별자로 사용하는 get(keyName), set(keyName, value) 메서드 사용
  • 2차원 배열 형태와 비슷한 구조
  • key의 타입은 문자열로 하는것이 원칙
  • key의 타입은 문자로 하는것이 원칙 => key의 타입이 달라지면 관리가 어려워지기 때문
  • set(key, value) : 키에 해당하는 값 변경
  • get(key) : key에 해당하는 value 리턴
  • delete(key) / has() / forEach() / entries() / size

DOM(Document Object Model) : 문서(html) 객체 모델

  • ''의 하위 태그는 각각의 기능(Method), 속성(properties)을 갖고있는 문서객체
  • elements 가져오기 : byID, byClassName, byTagName ...
  • document.getElementById('[id명]') : 단일 element 리턴
  • document.getElementByClassName('[class명]') : 복수 elements 리턴 => HTMLCollection 타입으로 리턴
  • document.getElementByTagName('[tag명]') : 복수 elements 리턴 => HTMLCollection 타입으로 리턴
  • getElements로 가져오는 객체는 우선적으로 HTMLCollection의 index를 선택해야함

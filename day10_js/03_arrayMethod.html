배열 관련 메서드 : 유사(연관)배열에서는 사용 제한이 있음.

  • join() : 배열원소 사이에 문자를 삽입하여 새로운 배열 생성
  • array.push(value) : 배열의 마지막에 value값을 추가 => array[array.length]=value 같은 의미
  • push() : 배열의 마지막에 valuer값 추가 후, 배열의 길이 리턴
  • pop() : 배열의 마지막 원소 제거 후, 제거한 원소값 리턴
  • shift() : pop과 위치 반대, 배열의 첫 원소 제거 후, 제거한 원소값 리턴
  • unshift(value) : 배열의 첫번째에 원소(value) 추가 후, 배열 길이 리턴

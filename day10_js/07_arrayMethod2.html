배열의 순환 할 수 있는 구조를 이용하여 특정 기능을 할 수 있는 메서드 제공

  • array.reduce(callback function(total, value, index, self), [init value]) : 연산의 누적값 리턴
  • array.reduceRight(callback function(total, value, index, self), [init value]) : reduce와 동일하지만 값을 우측에서 좌측으로 누적연산
  • array.every(callback function(value, index, self)) : // .every = and 콜백 함수의 리턴조건에 모든 요소가 만족하면 true, 아니면 false
  • array.some(callback function(value, index, self)) : // .some = or 콜백 함수의 리턴조건에 하나라도 만족하면 true, 아니면 false
  • array.indexOf("str", [start index]) : 해당 문자열이 존재하는 배열 idex를 리턴 (앞에서부터)
  • array.lastIndexOf("str", [start index]) : 해당 문자열이 존재하는 배열 idex를 리턴 (뒤에서부터)
  • array.find(callback function(value, index, self)) : 검색 조건에 만족하는 첫번째 배열 값 리턴
  • array.findIndex(call back function(value, index, self)) : 검색 조건에 만족하는 첫번째 배열 인덱스 리턴
  • array.from('[Object명]') : 순서가 있거나, 순환구조를 갖고있는 객체를 배열로 변환
  • array.keys() : 배열이 사용중인 key(식별자, index)를 검색하여 새로운 배열로 리턴 (객체에는 적용안됨)
  • array.entries() : index:value의 쌍 구조의 값을 만들어 새로운 배열에 담아 리턴 (객체에는 적용안됨)
  • array.includes('찾고싶은 값') : 배열에 찾는 값이 존재하면 true 리턴
